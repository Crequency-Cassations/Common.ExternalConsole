# v2 Release
我们改进了 Common.ExternalConsole!
使用 TCP 重写了网络通信相关逻辑.
同时提供了更好的提示符支持.

# 关于
这是一个外置的控制台交互方案  
您可以将该程序集成于您的项目中  
您可以使用 Common.ExternalConsole 来为您的控制台项目增加更多的交互窗口, 以便异步执行任务  
也可以为您的图形界面程序增加新的交互方式, 即使用命令行的形式来与您的程序交互
